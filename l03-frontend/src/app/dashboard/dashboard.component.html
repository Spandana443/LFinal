Technical Aspects

This project was developed using modern web development technologies:
  • Frontend: Angular with standalone components for modularity and dynamic rendering.
  • Backend: Node.js with JWT-based authentication to secure routes.
  • Database: MySQL for data persistence.
  • Charting Library: Chart.js for visualizing dynamic data.
  • Hosting: NGINX serves both the backend (on port 3000) and the frontend (on port 80).
  • Routing: SPA structure ensures seamless navigation.

Accessibility features (ADA/WCAG principles) have been incorporated into the frontend to ensure usability for a broader audience. The application is fully decoupled, allowing flexibility for future enhancements.
